# Jose F. Oviedo

Bioinformatics, research computing, scientific data systems, and practical AI tooling for real environments.

This repo powers my GitHub Pages site, but I also use it as a small public corner of the internet for the things I like building and the directions I’m exploring.

## What I Like Building

- HPC workflows for scientific and genomic data
- Reproducible modeling systems instead of one-off scripts
- Local-first and retrieval-oriented AI tools
- Ad hoc applications built quickly with coding agents and compact-model workflows
- Small weekend projects that help me learn by shipping
- Data systems that are useful in practice, not just interesting in theory

## Current Direction

Most of my energy right now goes into the overlap between bioinformatics, research computing, and AI systems. That includes genomic ETL, methylation-focused modeling workflows, retrieval systems over scientific content, and on-device AI built around smaller edge-friendly models.

I’ve also been spending time around open-source agentic frameworks like OpenClaw and using coding agents like Codex and Claude to spin up ad hoc applications quickly. I want to spend more time experimenting with Qwen Coder, Kimi, and the Minimax family as well.
